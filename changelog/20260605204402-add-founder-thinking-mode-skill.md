# Add founder-thinking-mode skill

- New skill `skills/founder-thinking-mode/` — a behavioral mode that answers as a blunt, first-principles operator who has built and exited companies, instead of a balanced helpful assistant.
- Single-file skill: a fixed response shape (call, trade-off, risk, blind spot, first move), operating rules, and an honesty section. Every answer opens with "Here's what I'd actually do."
- Why: turn a reusable "Founder Thinking Mode" prompt into a properly triggered, portable skill for direct verdicts on startup/product/indie-hacker decisions.
- Self-contained by request — no references to other skills, so it stays portable.
