# Add setup-adrs skill

- Added `setup-adrs` skill: bootstraps an Architecture Decision Record system in any project (ADR dir + template + seed ADR-0001, `ARCHITECTURE.md` recap doc, and an ADR policy injected into AGENTS.md/CLAUDE.md).
- Added its row to the README Skills table.
- Why: make the ADR pattern (decision records + recap + policy) reusable across repos, the same way `changelog-setup` bootstraps changelogs.
