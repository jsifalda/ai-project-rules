# Add first-principles-mode skill

- New skill `skills/first-principles-mode/` — a reasoning mode that strips a problem to fundamental truths, triages every assumption (verified / unverifiable / false), rebuilds the answer from only the verified set, and names where conventional wisdom is wrong.
- Why: operationalizes a recurring user need — reason from first principles on demand instead of defaulting to the conventional answer.
- Single-file instructional skill (no scripts/references), matching the repo norm for reasoning-mode skills like `grill-me` and `prompt-enhancer`.
